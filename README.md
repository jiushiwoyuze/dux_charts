# ONE 数据可视化图表库

[百度内网查看地址](http://youcanyouup.info/dux_charts/)

### 视觉设计师教程：
1. 挑选一个适合的图表；
2. 调整浏览器窗口，让图表宽度缩放，找到合适的引用宽度；
3. 右键“图片存储为”下载图表为一张 PNG 图片，应用到设计稿中占位；
4. 右键“复制”后黏贴在设计稿中使用；


### 交互设计师教程：
1. 挑选一个适合的图表；
2. 点击图表下方的“交互引用”链接，新窗口中打开纯图表；
3. 复制该纯图表页面的 URL 链接地址；
4. 在 Axure 中拖拽一个 IFrame 组件，填入以上 URL 链接地址；
